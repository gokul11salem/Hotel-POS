# Hotel-POS
Order and Billing POS for Hotel, Bakeries...

customizable web-based Point of Sale (POS) billing system built for tea stalls, cafes, tiffin centers, and small restaurants.

# 📌 Overview
Hotel POS is a single-page web application designed to simplify daily order processing, table management, bill printing, and sales tracking. Built with zero external framework dependencies, it runs entirely inside the browser using modern standard web APIs and standard HTML/CSS/JavaScript.

# ✨ Key Features
📱 Responsive & Intuitive Interface

Clean UI optimized for both desktop browsers and mobile/tablet devices.  

Light & Dark theme toggle for comfortable viewing in any environment.  

🍵 Interactive Shop & Catalog

Categorized menu layout (Beverages, Snacks, Tiffin, Parotta Varieties, Sweets, etc.).  

Real-time item count, quantity management, and slide-out cart drawer.  

🪑 Table & Order Management

Assign orders to customizable table numbers or mark as takeaway.  

Hold / Dine-In Orders: Save active dine-in orders and update or finalize them later.  

🖨️ Bill Printing & Summary

Custom thermal print layout for receipts (window.print() optimized).  

Auto-calculates total order value with multi-currency support.  

💳 Flexible Payment Integration

Supports custom UPI QR Code upload for quick customer scanning.  

Display bank account and direct payment transfer details directly at checkout. 

📊 Dashboard & Sales Analytics

Live snapshot of daily sales, total orders, monthly revenue, and pending dine-in orders.  

Export complete sales history into CSV format for accounting and record-keeping. 

🔒 PIN-Protected Admin Settings

Secure management portal protected by a 4–6 digit PIN (Default PIN: 1234).  

Customize shop name, slogan, logo, product catalog, pricing, bank details, and table layouts.  

All configurations and transaction histories persist locally via Browser LocalStorage. 

🚀 Getting Started
Since this project is completely client-side, running it requires no server setup or node dependencies!

Option 1: Direct File Usage
Clone or download this repository.

Bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
Open index.html in any web browser.

Option 2: Deploy to GitHub Pages
Push index.html to your GitHub repository.

Go to Settings > Pages in your GitHub repository.

Select the main branch as the source and click Save.

Access your live application link provided by GitHub Pages!

⚙️ Default Configuration
Default Admin PIN: 1234 (Can be changed under Settings ➔ Security)

Data Storage: All menu customizations, active sales, and setting updates are saved automatically in your browser's localStorage.
